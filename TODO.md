# 基本
- [x] GETリクエストをしてHTMLを取得する
- [ ] 取得したHTMLから最初のスライドのURIを取得する
- [ ] 現在のスライドURIから次のスライドのURI取得する
- [ ] Serverを立ち上げて、スライドが表示されるようにする
- [ ] 次へのボタンを押すと、次のスライドが表示されるようにする
- [ ] テストを書く。以降は、テストを追加
- [ ] 最大スライド数を取得する
- [ ] 最大スライド数よりは先に進めないようにする
- [ ] 戻るボタンを押すと、前のスライドが表示されるようにする
- [ ] 1ページより前には、戻れないようにする
